### 构建 

在 MINGW 环境中编译，确保安装了 MINGW 下 GCC 构建工具链。

```bash
# 仅编译
make # 或 mingw32-make
# 编译加运行
make run
```
如遇到问题，也可直接使用`run.bat`编译运行，但编译速度较慢。
